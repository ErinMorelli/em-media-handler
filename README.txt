# EM Media Handler #

*by Erin Morelli*

A comprehensive media handler for use with Deluge to manage TV Shows, Movies, Music, and Audiobooks. Integrates email and push notifications via Pushover.


### Packages & Apps Used ###

* [Deluge](http://deluge-torrent.org/)
* [Filebot](http://www.filebot.net/)
* [Beets](http://beets.radbox.org/)
* [ABC](http://www.ausge.de/ausge-download/abc-info-english)
* [Google Books API](https://developers.google.com/api-client-library/python/apis/books/v1)
* [Mutagen](https://bitbucket.org/lazka/mutagen)
